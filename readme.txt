
Breakout
Breakout is a classic arcade game written in Python. The objective of the game is to break all the bricks on the screen by bouncing a ball off a paddle controlled by the player. 

Requirements
Python 3.x
Pygame library

How to play
Use the left and right arrow keys to move the paddle and keep the ball in play
Break all the bricks on the screen to win the game
